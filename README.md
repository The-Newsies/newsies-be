# :newspaper: Newzie :newspaper:

This is the back-end repo for Newzie, an app that allows you to consume and organize news from a variety of sources.

## The Tech

Back-end
* MongoDB/Mongoose
* Node
* Express
* Jest
* Hosted on Heroku
